# 搜索测试文档

## 第一章 介绍

这是一个用于测试搜索功能的文档。文档中包含多个段落，每个段落都有不同的关键词。

搜索功能支持正文搜索、目录搜索和批注搜索三种模式。

## 第二章 功能说明

### 2.1 正文搜索

正文搜索使用 Ctrl+F 快捷键打开搜索栏。输入关键词后，文档中所有匹配的文本会被高亮标记。

用户可以通过上下导航按钮在匹配项之间切换。当前匹配项会有特殊的高亮样式。

### 2.2 目录搜索

目录面板顶部有一个搜索框，输入关键词后目录列表会被过滤，只显示匹配的条目。

过滤时会保持层级结构，即如果子项匹配，其父项也会显示。

### 2.3 批注搜索

批注面板也支持搜索功能，可以按批注内容、类型等进行过滤。

## 第三章 技术实现

搜索功能使用 TreeWalker API 遍历文档 DOM 树，找到所有文本节点中的匹配项。

匹配项使用 `<mark>` 标签包裹，添加 `search-highlight` 类名实现高亮效果。

## 总结

本文档用于验证搜索功能的正确性，包含足够多的文本内容和层级结构。
