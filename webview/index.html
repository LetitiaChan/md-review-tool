📖 使用帮助

🚀 快速开始

在 VSCode 中打开 .md 或 .mdc 文件，通过以下方式打开批阅面板：

  • 命令面板：Ctrl+Shift+P → 搜索「MD Human Review: 打开批阅面板」
  • 右键菜单：在编辑器或资源管理器中右键选择
  • 编辑器标题栏：点击标题栏图标按钮

💬 添加评论

  1. 在文档中选中一段文字
  2. 松开鼠标后出现浮层工具条，点击 「💬 评论」（也可右键选择）
  3. 在弹窗中输入评论内容，可选上传图片（点击、拖拽或 Ctrl+V 粘贴）
  4. 点击 「提交评论」

选中的文字会被标记为 紫色高亮，右侧批注栏同步显示评论卡片。

🗑️ 标记删除

  1. 选中需要删除的文字
  2. 点击浮层中的 「🗑️ 删除」（或右键选择）

被标记的文字会以 删除线 样式展示。

➕ 插入内容

  1. 选中一段文字作为插入位置的锚点
  2. 右键选择 「➕ 插入内容（在此处之后）」 或 「⬆️ 插入内容（在此处之前）」
  3. 输入要插入的内容（支持 Markdown 格式），可选填插入说明

✏️ 直接编辑（所见即所得）

  1. 点击工具栏的 「预览/编辑」 切换按钮（或按 Ctrl+Shift+E）
  2. 直接在渲染后的文档上修改内容，支持 WYSIWYG 工具栏（加粗、斜体、删除线、标题、列表、引用、分隔线、撤销/重做）
  3. 编辑模式下右键表格可 插入/删除行列
  4. 按 Ctrl+S 手动保存，或等待自动保存

🤖 一键 AI 修复

  1. 完成批阅后，点击工具栏 「一键AI修复」 按钮
  2. 确认后生成结构化 AI 修改指令文件，保存到 批阅文件/ 目录
  3. 点击 「🚀 确定执行」 自动发送指令到 CodeBuddy / 工蜂 Copilot 对话窗口
  4. 也可点击 「📋 一键复制指令」 手动粘贴到任意 AI 工具

📤 导出与存储

  • 按 Ctrl+E 导出批阅记录为 AI 可读的结构化 Markdown 指令文件
  • 批注记录 自动保存 到工作区 .review/ 目录，批注清空时自动删除记录文件
  • 源文件内容变更时自动归档旧版本，创建新批阅版本

🖥️ 界面功能

  • 目录导航 — 自动生成文档目录，支持折叠/展开、快速跳转、滚动高亮当前章节
  • 批注面板 — 侧边批注列表，支持按时间或文本位置排序、定位、编辑、删除
  • 图片灯箱 — 点击图片放大预览，支持滚轮缩放、拖拽平移、双击还原（+/- 缩放，0 适应窗口，R 重置）
  • Mermaid 图表 — 点击 Mermaid 图表放大查看，支持缩放控制条
  • 禅模式 — 按 Alt+Z 隐藏侧栏专注阅读，同时隐藏 IDE 侧边栏
  • 亮色/暗色主题 — 工具栏一键切换或跟随系统
  • 文件选择器 — 工作区内 Markdown 文件下拉框快速切换
  • 文件变更检测 — 源文件修改后显示「文件已更新」徽章，点击刷新按钮重载
  • 面板拖拽 — 目录和批注面板宽度可拖拽调整
